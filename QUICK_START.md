# Quick Start Guide - MindCare

## Setup in 3 Easy Steps

### 1️⃣ Install Dependencies
```bash
npm install
```

### 2️⃣ Start the App
```bash
npm run dev
```

The app will open at `http://localhost:5173`

### 3️⃣ Get Your Free Groq API Key

1. Go to https://console.groq.com
2. Sign up (it's free!)
3. Navigate to "API Keys"
4. Click "Create API Key"
5. Copy the key
6. In the app, click "Settings" tab
7. Paste your API key and click "Save"

## First Time Use

1. **Start Tracking**: Go to "Track Mood" tab and log your first mood entry
2. **View Analytics**: After a few entries, check out the "Analytics" tab to see your patterns
3. **Try AI Support**: Click "AI Support" and then "Get Insights" for personalized recommendations
4. **Chat with AI**: Ask questions or discuss how you're feeling

## Tips

✅ **Track Daily**: Try to log your mood at the same time each day for better insights

✅ **Be Honest**: The more honest you are, the better the AI can support you

✅ **Use Activities**: Tagging activities helps identify patterns

✅ **Add Notes**: Personal notes provide context for your mood

✅ **Review Analytics**: Check your trends weekly to understand patterns

## Privacy & Security

🔒 All your data stays in YOUR browser
🔒 API key stored securely in localStorage
🔒 No data sent to third-party servers (except Groq for AI features)
🔒 You can clear data anytime via browser settings

## Troubleshooting

**AI not working?**
- Make sure you've added your Groq API key in Settings
- Check that the API key is valid at console.groq.com

**Data disappeared?**
- Data is stored in browser localStorage
- Clearing browser data will delete your entries
- Use the same browser/device to see your data

**Charts not showing?**
- Add at least 2-3 mood entries to see analytics
- Charts update automatically when you add new entries
